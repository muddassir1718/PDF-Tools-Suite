## 🔒 গোপনীয়তা এবং নিরাপত্তা (Privacy First)
সাধারণ অনলাইন পিডিএফ কনভার্টারগুলো আপনার ফাইল তাদের সার্ভারে আপলোড করে প্রসেস করে, যা গোপনীয় ডেটার জন্য অত্যন্ত ঝুঁকিপূর্ণ। **PDFTools Suite**-এ কোনো ফাইল সার্ভারে পাঠানো হয় না। ফাইল রিডিং এবং কনভার্সন বা প্রসেসিং সম্পূর্ণ ইভেন্ট-রিভেন আর্কিটেকচারের মাধ্যমে আপনার লোকাল ব্রাউজারে ঘটে। ব্রাউজার ট্যাব বন্ধ করার সাথে সাথে মেমোরি সম্পূর্ণ খালি হয়ে যায়।

## 🛠️ সুনির্দিষ্ট ফিচারসমূহ (Core Features Breakdown)

### ১. রূপান্তরকারী বা কনভার্ট গ্রুপ (Convert PDF Group)
* 📘 **PDF to Word:** পিডিএফ ফাইলের টেক্সট লেআউট স্ক্যান করে সরাসরি এডিটেবল ওয়ার্ড ডক্স (`.docx`) ফাইলে রূপান্তর করুন।
* 📙 **Word to PDF:** আপনার তৈরি ওয়ার্ড (`.docx`) ফাইল আপলোড করে ম্যামথ পার্সার ইঞ্জিনের মাধ্যমে নিমিষেই নিখুঁত পিডিএফে রূপান্তর করুন।
* 📗 **PDF to Excel:** পিডিএফে থাকা কোনো টেবিল বা সারিবদ্ধ ডেটা সনাক্ত করে সরাসরি মাইক্রোসফট এক্সেল বা স্প্রেডশিট (`.xlsx`) ফাইলে সাজিয়ে নিন।
* 📙 **PDF to PowerPoint (PPT):** পিডিএফ-এর প্রতিটি পৃষ্ঠাকে হাই-কোয়ালিটি ১৬:৯ ওয়াইডস্ক্রিন পাওয়ারপয়েন্ট স্লাইডে রূপান্তর করুন।
* 📕 **PDF to JPG:** পিডিএফ-এর পেজগুলোকে হাই-রেজোলিউশন ইমেজ জিপ ফরম্যাটে ডাউনলোড করুন।
* 📒 **JPG to PDF:** জেপিজি ফটোগুলো একের পর এক সাজিয়ে ও পুনর্বিন্যাস করে একটি একক পিডিএফে রূপান্তর করুন।
* 🌸 **Image to PDF:** ব্যাচ আকারে ওয়ান-ক্লিক পোর্টেবল ডকুমেন্ট ফরম্যাটে রূপান্তর করুন (PNG, WEBP, GIF ও JPG সাপোর্ট করে)।
* 📄 **PDF to Text:** পিডিএফ ফাইলের সব সিলেক্টেবল লেখা এক্সট্র্যাক্ট করে কপি অথবা সরাসরি `.txt` ফাইল হিসেবে ডাউনলোড করুন।

### ২. সাজানোর ইউটিলিটি গ্রুপ (Organize PDF Group)
* 🔗 **Merge PDF:** একাধিক পিডিএফ ফাইল আপলোড করে মাউস দিয়ে ড্র্যাগ করার মাধ্যমে সেগুলোর ক্রম সাজিয়ে একটি একক ফাইলে রুপান্তর করতে পারবেন।
* ✂️ **Split PDF:** একটি পিডিএফ থেকে সুনির্দিষ্ট কোনো পেজ রেঞ্জ (যেমন: ১-৫) বা পৃথক পেজ এক্সট্র্যাক্ট করতে কিংবা প্রতি N পৃষ্ঠা পর পর বিভক্ত করতে পারবেন।
* 📉 **Compress PDF:** ফাইলের মূল চমৎকার ভিজ্যুয়াল কোয়ালিটি বজায় রেখে পিডিএফ-এর সাইজ অনেক কমিয়ে ফেলতে সাহায্য করে। এখানে লো, মিডিয়াম ও হাই ৩টি লেভেলে কম্প্রেস করা সম্ভব।
* 🔓 **Unlock PDF:** পিডিএফ ফাইলে থাকা প্রিন্টিং বা কপি করার সীমাবদ্ধতা ও পাসওয়ার্ড প্রটেকশন সহজেই দূর করে ক্লায়েন্ট-সাইড মেমোরিতে নতুন ফাইল সেভ করুন।

# PDFTools Suite 📄⚡ (English Version)

A secure, fast, and 100% client-side in-browser PDF utility platform where your files are parsed completely on your local computer CPU.

## 🔒 Privacy-First Design
Unlike standard utilities that upload your tax returns, financial statements, or personal records to a background queue, our application executes **strictly in-browser**. All documents are structured inside browser memory layers and garbage-collected once the tab is closed.

## 🛠️ Core Features

### 🔄 Conversion Tools
- **PDF to Word:** Reconstruct raw printable structures into Microsoft Word (`.docx`) file documents.
- **Word to PDF:** Local formatting and generation of clean `.pdf` containers from uploaded `.docx`.
- **PDF to Excel:** Vertical column and row baseline grid mapping to generate direct spreadsheets (`.xlsx`).
- **PDF to PowerPoint (PPT):** Layout renderers to widescreen 16:9 slides compiled onto interactive presentation bundles.
- **PDF to JPG:** Convert page layout assets to high-density `.jpg` photos organized into a convenient `.zip` format.
- **JPG to PDF:** Simple photos compiler supporting ordered alignments.
- **Image to PDF:** Batch container wrapping JPG, PNG, WEBP, and GIF frames.
- **PDF to Text:** Simple raw plain-text extraction to `.txt` or direct copy-paste.

### 🛠️ Organizing Features
- **Merge PDF:** Multi-file collector supporting layout drag shifts.
- **Split PDF:** Segment pages by custom ranges (e.g., 2-4), extract single templates, or split files by increments.
- **Compress PDF:** Multi-option scale optimizer (Low/Medium/High compression rates).
- **Unlock PDF:** Remove locks, copying constraints, and security banners with standard local password variables.

## ⚡ Key Benefits
- **Zero Registration:** No premium layers, subscriber locks, or payment constraints.
- **Local Speeds:** Standard zero-latency workflow bypassing network queues completely.
- **Robust Touch Interactions:** Ideal for modern mobile and tablet devices.
- **Integrated Themes:** Native high-contrast Dark and Light mode transitions.

## 💻 Local Installation & Setup

If you wish to host this application locally on your desktop or machine, follow these simple steps:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/yourusername/pdftools-suite.git
   cd pdftools-suite
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run the Development Server:**
   ```bash
   npm run dev
   ```
   Open your browser to `http://localhost:3000` to view the application.

4. **Build Production Bundles:**
   ```bash
   npm run build
   ```
